# **EPUB TO PDF** _(WITH TRANSLATION)_
---
> ## **_What the project does?_**

The project main usage is to convert a .epub file in a .pdf file.
The second thing, is to translate the epub file into a chosen language, trying to maintain the same layout of the sheet page.
---
> ## **_Why the project is useful_**

Like I Said before, the project want to help people translate an ebook, in their own languages.
---
> ## **_ABOUT_**

I'm a Chess Player, so the reason behind this project was to translate ebook in my own language, trying not to touch the moves of chess pieces.
